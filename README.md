# Free AI Social Media Scheduler

A free, open-source AI social media scheduler built with Next.js. Schedule posts, generate AI content, and manage all your social accounts — no subscription required.

> **Looking for AI-generated images for your social media posts?** Check out **[Awesome GPT-Image-2 Prompts](https://github.com/Anil-matcha/Awesome-GPT-Image-2-API-Prompts)** — a curated collection of prompts for generating stunning social media visuals.

---

## What is this?

A self-hostable alternative to tools like Postiz, Buffer, and Hootsuite — with built-in AI to help you generate captions, images, and post schedules automatically.

## Features (Coming Soon)

- **AI Caption Generator** — generate platform-optimized captions for Instagram, LinkedIn, Twitter/X, and more
- **AI Image Generation** — create scroll-stopping visuals using GPT-Image-2 directly from the scheduler
- **Multi-platform Scheduling** — schedule posts across all major social networks from one dashboard
- **Content Calendar** — visual calendar view of all scheduled and published posts
- **Analytics** — track engagement, reach, and best posting times
- **Self-hostable** — run it on your own server, full control over your data

## Tech Stack

- **Framework:** Next.js
- **AI:** OpenAI GPT-Image-2 + GPT-4o
- **Database:** PostgreSQL
- **Auth:** NextAuth.js
